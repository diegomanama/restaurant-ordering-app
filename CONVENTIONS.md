# Conventions

**Purpose:** quick reference for filename, export, and import rules used in this repo.

## File naming

- General modules / utilities: **kebab-case**
- Single-export modules (e.g. `data.js`, `addEventListenerAll.js`): **camelCase**

## Module exports

- Prefer **named exports** for everything. I don't like default exports :v

## Imports

- Prefer absolute paths for imports.
